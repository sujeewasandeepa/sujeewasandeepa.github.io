Hello there folks,
This is my first real post on this blog site, I made with jekyll on github pages. First I want to make sure that this is
not a perfect site or a perfect post, this is a work in progress. So please forgive me and support if you saw I do something 
stupid. 

If you didn't know that you could make a website and host it free of charge, I think you will be pretty amazed and happy too. 
Specially if you are from a third world ever developing country like me. So this article is for those people well, but not 
exclusively. And I will do my best to explain every single step in the simplest form possible. Because I know this can be a little hard if you haven't done something like this before, haha yes I know that by experience. I had to play around this a whole day to figure out most of the things. 

So yes enough chit chat and let's see what we need first,
1. First you need to create a github account, I won't explain how to do that. Because it's just too easy.
2. I personally don't like proprietory software, but the world is advancing too fast and we are forced to kiss arses of big corporations, so for this article I will use visual studio code because with it it's just so damn easy. So yes if you do not have installed visual studio code on your computer, you need to install it. It's available on Windows, Mac and Linux.
3. Okay now log into your github account, and then you need to create a new repository. Not just a typical repository, this repository should like this, "\<your username>.github.io"
4. And you need to install jekyll on your machine, it is not hard. But if you came up with any issue I would recommend you to follow this <a href="https://jekyllrb.com/docs/installation/">documentaion</a>.
